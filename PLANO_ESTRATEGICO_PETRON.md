### A equacao que nao fecha

```
R$ 75.000 MRR ÷ 53 clientes = R$ 1.426/cliente

Para entregar:
- Trafego pago (Meta + Google)
- Conteudo organico (posts, carrosseis, reels)
- Design grafico
- Producao de video
- Customer Success
- Captacao de imagens
- Social Media Management
- SDR + Closer (comercial)

= Operacao de ~R$ 150k/mes de complexidade
  com ~R$ 75k/mes de receita
```

### O dilema da identidade

A Petron opera como **duas empresas ao mesmo tempo:**

| Empresa A (MatCon) | Empresa B (Legado) |
|--------------------|--------------------|
| 26 clientes | 27 clientes |
| Vertical clara, ICP definido | Nichos dispersos, sem padrao |
| Metodo Petron aplicavel | Cada cliente e uma excecao |
| Escala via padronizacao | Cada escopo e artesanal |
| Dados comparaveis entre clientes | Sem benchmark possivel |
| Potencial de ticket alto | Ticket estagnado |

**Diagnostico central:** A agencia esta presa entre dois modelos. O modelo MatCon tem escala, o modelo legado consome margem. A decisao estrategica e **escolher um e proteger o outro com regras claras.**

## 1.4 Os 7 Gargalos Que Travam o Crescimento

### Gargalo 1: Ticket medio baixo (R$ 1.426) para complexidade alta
- **Causa:** Planos Start a R$ 997, Conteudo Bronze a R$ 800, sem reajuste por complexidade real
- **Efeito:** Margem comprimida, equipe sobrecarregada, sem caixa para investir
- **Meta:** Ticket medio R$ 2.000+ ate dez/2026, R$ 2.500+ ate dez/2027

### Gargalo 2: Clientes legados de social media contaminando modelo
- **Causa:** Clientes de Conteudo Bronze/Prata fora do nicho pagam R$ 800-1.300/mes por um servico que consome tempo de SM + designer + captacao
- **Efeito:** Designers e SM gastam tempo em nichos que nao geram benchmark, nao geram upsell, nao geram case
- **Meta:** Zero clientes Bronze fora do nicho ate set/2026

### Gargalo 3: Joao com 26 clientes (130% da capacidade)
- **Causa:** Crescimento organico sem contratacao + Talita fazendo CS + trafego
- **Efeito:** Qualidade cai, tempo de resposta aumenta, burnout
- **Meta:** Maximo 22 clientes por gestor ate jun/2026

### Gargalo 4: Felipe no limite (115h/mes = 96% capacidade)
- **Causa:** Videomaker unico para 53 clientes + captacoes
- **Efeito:** Qualquer novo cliente com video trava a operacao
- **Meta:** Reduzir para 80h/mes via IA (roteiro, edicao assistida) ou contratar freelancer

### Gargalo 5: Designers com erros de portugues e retrabalho
- **Causa:** Sem validacao automatica, sem gate de qualidade antes do "done"
- **Efeito:** SM precisa revisar cada legenda, retrabalho diario
- **Meta:** Spell check IA obrigatorio antes de marcar post como concluido (mai/2026)

### Gargalo 6: Aprovacao de conteudo 100% manual
- **Causa:** Fluxo Canva→exporta PDF→envia WhatsApp→recebe feedback por audio→transcreve→aplica
- **Efeito:** ~45min por cliente por mes = ~40h/mes da equipe
- **Meta:** Portal de aprovacao com link publico ate set/2026

### Gargalo 7: Zero inteligencia de dados do nicho consolidada
- **Causa:** 26 clientes MatCon = dados de 26 mercados diferentes, nunca cruzados
- **Efeito:** Perde a oportunidade de construir benchmark proprietario (CPL por regiao, oferta que mais converte, sazonalidade por categoria)
- **Meta:** Benchmark MatCon v1 ate dez/2026

# PARTE 2 — A TESE: PARA ONDE VAMOS

## 2.1 Posicionamento Futuro

### Mudanca de Posicionamento

| | ANTES (2024-2026) | DEPOIS (2026-2027) |
|---|---|---|
| **Proposta** | "Gerimos seu trafego e redes sociais" | "Aumentamos demanda local e vendas previsiveis para lojas de MatCon com operacao padronizada, dados do setor e IA" |
| **Nicho** | Qualquer comercio local | Lojas de materiais de construcao |
| **Modelo** | Agencia de servico por hora | Operacao de crescimento verticalizada |
| **Diferencial** | Preco | Inteligencia do nicho + tecnologia proprietaria |
| **Entrega** | Posts + anuncios | Conversas qualificadas no WhatsApp → orcamentos → vendas |
| **Pricing** | Fee fixo por escopo | Fee base + bonus por resultado (onde rastreavel) |
| **Escala** | Contratar mais gente | Padronizar + automatizar + empilhar dados |

### O Metodo Petron como produto

O Metodo Petron ja existe informalmente:
1. **Mercado** — analise de raio, concorrencia, populacao, poder de compra
2. **Oferta** — boi de piranha (produto ancora) + mix de acabamento
3. **Trafego** — campanha Meta Ads otimizada para conversas no WhatsApp

O que muda: esse metodo vira **o produto padrao**, aplicado a todos os clientes MatCon com a mesma sequencia, mesmos KPIs, mesmos criativos de referencia. Clientes fora do nicho nao recebem o Metodo Petron — recebem escopo fechado sem personalizacao.

## 3.1 Criterios de Scoring (1-5 cada)

| Criterio | Peso | 5 pontos | 3 pontos | 1 ponto |
|----------|------|----------|----------|---------|
| **Fit com MatCon** | 30% | Loja MatCon pura | Adjacente (moveis, eletricos) | Totalmente fora (pet, moda, educacao) |
| **Margem real** | 25% | Fee > R$ 2.500 com escopo simples | Fee R$ 1.500-2.500 | Fee < R$ 1.000 ou escopo inflado |
| **Complexidade operacional** (inverso) | 20% | So trafego, sem conteudo organico | Trafego + conteudo padrao | Trafego + conteudo + captacao + excecoes |
| **Potencial de expansao** | 15% | Pode subir de plano, tem multiplas lojas | Pode subir de plano | Ja no maximo do que faria sentido |
| **Saude do relacionamento** | 10% | Promotor, paga em dia, facil de trabalhar | Neutro | Detrator, atrasa, demanda excessiva |

### Formula de scoring

```
Score = (Fit x 0.30) + (Margem x 0.25) + (Complexidade_inversa x 0.20) + (Expansao x 0.15) + (Relacionamento x 0.10)
```

| Aspecto | Detalhe |
|---------|---------|
| **Antes** | Talita gere 8 clientes de trafego + faz CS |

# GLOSSARIO

| Termo | Definicao |
|-------|-----------|
| **MatCon** | Loja de materiais de construcao |
| **Boi de piranha** | Produto com preco agressivo para atrair trafego (ancora) |
| **CPL** | Custo por lead (conversa no WhatsApp) |
| **CPA** | Custo por aquisicao (orcamento qualificado) |
| **MRR** | Monthly Recurring Revenue (receita recorrente mensal) |
| **CAPI** | Conversions API (Meta) — tracking server-side |
| **Fee** | Mensalidade cobrada do cliente |
| **ICP** | Ideal Customer Profile (perfil de cliente ideal) |
| **LTV** | Lifetime Value (valor de vida do cliente) |
| **Churn** | Cancelamento de cliente |
| **NPS** | Net Promoter Score (indicador de satisfacao) |
| **Raio de atuacao** | Area geografica que a loja atende |
| **Curva ABC** | Classificacao de produtos por volume de venda |
| **Acabamento** | Linha de maior margem (porcelanatos, loucas, metais, tintas premium) |
